One of the most challenging aspects of learning HTML, CSS, and JavaScript was understanding how the different languages work together to create a website. At first, it was difficult to remember the syntax and understand how small mistakes in code could affect the entire page. CSS was especially challenging when trying to position elements correctly and make the design look organized. JavaScript was also difficult in the beginning because it introduced functions, events, and interactive features that required more problem-solving.

I overcame these challenges by practicing regularly and experimenting with my code. When something did not work, I tried to troubleshoot the problem step by step and test different solutions. Over time, this helped me become more comfortable reading and fixing errors. I also improved by building small projects and gradually adding more features as my confidence increased.

The most useful resources for me were classroom examples, online tutorials, coding practice, and feedback from my teacher and classmates. Using browser developer tools and reviewing examples of working code also helped me better understand how websites are structured and styled.
